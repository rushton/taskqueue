# Task Queue

A task manager for daily work and tasks. Work on one thing at a time, in order.

???

# Usage

```shell
#> tq wash the dishes
#> tq do the laundry
#> tq vaccum the house

#> tq
"wash the dishes"

#> tq done
Done: wash the dishes
Next: do the laundry

#> tq
do the laundry
```
